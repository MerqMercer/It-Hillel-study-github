Дан абстрактный класс шаурмы:

abstract class Shawarma
{
   /**
    * @return float
    */
   abstract public function getCost(): float;

   /**
    * @return array
    */
   abstract public function getIngredients(): array;

   /**
    * @return string
    */
   abstract public function getTitle(): string;
}
Метод getCost: возвращает цену шаурмы
Метод getIngredients: возвращает массив из ингредиентов шаурмы (строки)
Метод getTitle: возвращает название шаурмы

1. Создать по классу на каждый вид шаурмы, унаследовав базовый класс.
